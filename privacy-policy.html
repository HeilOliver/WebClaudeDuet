Privacy Policy

Last updated: January 2025

This privacy policy describes how the "Voice Input for Claude" Chrome extension ("the Extension") handles your data.

Summary: We do not collect, store, or transmit any of your personal data to our servers. All data stays on your device or is sent directly to OpenAI for transcription.

Data Collection

The Extension does not collect any personal information. Specifically:

  • We do not collect analytics or usage data
  • We do not track your browsing activity
  • We do not collect your conversations with Claude
  • We do not have any servers that receive your data

Data Storage

The Extension stores the following data locally on your device using Chrome's built-in storage API:

  • OpenAI API Key: Your API key is stored locally in Chrome's sync storage to enable voice transcription. This key is never transmitted to any server other than OpenAI's API.

Audio Recording

When you use the voice input feature:

  • Audio is recorded temporarily in your browser's memory
  • The audio is sent directly from your browser to OpenAI's Whisper API for transcription
  • Audio data is not stored permanently and is discarded after transcription
  • We do not have access to your audio recordings

Third-Party Services

The Extension uses the following third-party service:

  • OpenAI Whisper API: Your audio recordings are sent to OpenAI for transcription. OpenAI's privacy policy applies to this data processing. You can review OpenAI's privacy policy at openai.com/privacy.

Permissions

The Extension requests the following permissions:

  • storage: To save your OpenAI API key locally
  • activeTab: To inject the microphone button into the Claude interface
  • Host permission (claude.ai): To operate on the Claude website
  • Microphone (requested at runtime): To record your voice for transcription

Data Security

Your API key is stored using Chrome's secure storage API. We recommend:

  • Using an API key with appropriate usage limits
  • Monitoring your OpenAI API usage dashboard
  • Revoking and replacing your key if you suspect unauthorized access

Children's Privacy

The Extension is not intended for use by children under 13 years of age.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last updated" date above.

Contact

If you have questions about this privacy policy, please open an issue on our GitHub repository or contact us through the Chrome Web Store listing.
